# Rosalie Frazier Pazant Foundation Website

A Next.js website for the Rosalie Frazier Pazant Foundation — honoring a legacy of education and expanding pathways for Beaufort students.

## Pages

- **Home** (`/`) — Landing page with hero, about, mission & vision, core values, programs, impact, ways to support, founder's message, and contact
- **Biography** (`/biography`) — Full biography of Rosalie Frazier Pazant with timeline and milestones

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
